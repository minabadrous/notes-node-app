# notes-node-app
A note taking app width Node.js functionality connected directly to the File System
</br></br></br>
## List notes:
```
node list
```
## Add a note:
```
node add --title="[title]" --body="[body]"
```
## Read a note
```
node read --title="[title]"
```
## Remove a note
```
node remove --title="[title]"
```
